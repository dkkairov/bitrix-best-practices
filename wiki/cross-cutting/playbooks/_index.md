# Playbooks — пошаговые сценарии по жизненному циклу

Сквозные сценарии внедрения по этапам: пресейл/аудит → проектирование → настройка/деплой →
миграция данных → обучение и адаптация → поддержка.

## Этапы
- Пресейл и аудит
- Проектирование решения
- Настройка и запуск (деплой)
- Миграция данных
- Обучение и адаптация пользователей
- Поддержка и развитие

## Страницы
- [[presale-audit]] — playbook предпроектного аудита

## Статус покрытия
Есть аудит. Не хватает: деплой «под ключ», миграция данных, обучение, регламент поддержки.
