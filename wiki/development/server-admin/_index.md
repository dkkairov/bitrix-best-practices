# Администрирование сервера — практики (коробка)

Окружение `bitrix-env`, веб-сервер (nginx/apache + php-fpm), БД, резервное копирование, мониторинг,
безопасность сервера, проверка `bitrix_server_test`.

## Что покрываем
- Развёртывание окружения, рекомендуемые версии
- Бэкапы: стратегия, проверка восстановления
- Мониторинг, логи, алерты
- Безопасность: обновления, доступы, веб-антивирус, проактивная защита
